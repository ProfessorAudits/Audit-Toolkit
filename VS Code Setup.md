## Shortcuts Setup
```
- Ctrl + P : Opens the command Pallete, 
              Here, we can :Linenumber to go to exact line or ctrl+G to go to line number
              or @ to go to exact function or state variable
              or # to find in any folder or file or ctrl +t
              also we can open files from here

- ctrl+tab: to navigate through opened files in editors
- ctrl+shift+. : to list down all functions and state variables of the opened file

- ctrl + b: to open & close file explorer_ OR shift+e
- ctrl + d : after selecting_the_word to go to other places where it is used
- ctrl + p : go to file
- ctrl + shift + n : create new folder
- ctrl + n : new file

- ctrl+ shift+ g:  Source Control(github commits tab)
- ctrl+ shift + x: extensions
- ctrl+ shift + o : View all functions of contract
- ctrl+shift +[ : to minimize a function
- shift + o  : to view outline
- ctrl + alt + - : go backwards
- ctrl + shift + - : go forwards

```
